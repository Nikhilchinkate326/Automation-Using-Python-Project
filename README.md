# Corporate Data Analyzer

---

A Python-based desktop application that enables users to load Excel datasets, generate summarized reports, and visualize data through interactive charts. The application provides a simple graphical interface for performing basic data analysis without writing code.

---

##  Project Overview

Corporate Data Analyzer is a GUI application developed using Python that allows users to:

- Import Excel datasets
- View dataset information
- Generate summary reports
- Perform aggregations
- Create visual charts
- Export reports and charts

The project is designed for beginners and business users who want to analyze Excel data through an easy-to-use interface.

---

##  Features

-  Load Excel (.xlsx) files
-  Display dataset information (Rows, Columns, Column Names)
-  Generate reports using Group By operations
-  Perform aggregations:
  - Count
  - Sum
  - Average
  - Minimum
  - Maximum
-  Create charts:
  - Pie Chart
  - Bar Chart
  - Line Chart
-  Export generated reports
-  Export charts as images

---

## Technologies Used

- Python
- Pandas
- Tkinter
- Matplotlib
- OpenPyXL

---

##  Application Workflow

1. Browse and select an Excel dataset.
2. Read the dataset.
3. View dataset details.
4. Select:
   - Group By Column
   - Aggregation
   - Value Column
5. Preview the report.
6. Generate charts.
7. Export report or chart.
